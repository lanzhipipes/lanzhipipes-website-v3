# Lanzhi Pipes Website

Production-ready Next.js website for Lanzhi Pipes Technology Co., Ltd.

## Local preview

1. Install Node.js 18 or newer.
2. Run `npm install`
3. Run `npm run dev`
4. Open `http://localhost:3000`

## Deploy to Vercel

1. Upload this folder to a GitHub repository.
2. Import the repository into Vercel.
3. Vercel detects Next.js automatically.
4. After the preview is approved, add `lanzhipipes.com` and `www.lanzhipipes.com` in Vercel Domains.
5. Point the domain DNS to the values shown by Vercel.
6. Set one domain as the primary domain.

## Important

The inquiry form opens WhatsApp with the visitor's completed request. No email service key is required.
